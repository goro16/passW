# passW

passW is a CLI password manager for linux written in python3 with root authentication.

### Prerequisites :

- linux os
- python3
- root access


### Usage :

- Installation :

```
sudo apt install python3
python3 setup.py
```

- Usage :

```
python3 launch.py
```
